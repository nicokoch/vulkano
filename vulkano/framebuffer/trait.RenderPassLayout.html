Required Methods

fn convert_clear_values(&self, Self::ClearValues) -> Self::ClearValuesIter

Decodes a ClearValues into a list of clear values where each element corresponds to an attachment. The size of the returned array must be the same as the number of attachments.

The format of the clear value must match the format of the attachment. Only attachments that are loaded with LoadOp::Clear must have an entry in the array.
